  • Featured Skills
    Java SE, MySQL, JavaMail, GIT, Javadoc, Swing, AWT, MVC, Builder pattern, PDF generation.

    File Handling - reading / writing to txt file. PDF generation. E-Mail sending with SSL protocol.

The application allows to book a bus ticket. It shows you the bus layout and allows you to choose the seat. Afterwards the ticket is printed to pdf and confirmation is send to passengers e-mail address. All ticket info is archived in database and can be managed by GUI provided.
